Architecture

Not a prompt.
A pipeline.

01
Intent Extraction
Parses raw description into structured intermediate form — entities, roles, features, and relationships identified and normalized before any schema is generated.
STRUCTURED_INTERMEDIATE_FORM
02
System Design Layer
Converts extracted intent into full application architecture. Defines data flows, access control rules, and component relationships before touching any schema.
ARCHITECTURE_GRAPH
03
Schema Generation
Emits four synchronized outputs: UI config, API config, DB schema, and auth rules — guaranteed cross-layer consistent at generation time.
UI · API · DB · AUTH
04
Validation + Repair Engine
Detects invalid JSON, missing keys, hallucinated fields, schema mismatches. Repairs surgically — not full retry — targeting only the inconsistent layer.
DETERMINISTIC_REPAIR
